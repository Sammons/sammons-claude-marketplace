# sammons-claude-marketplace

Claude Code plugins by Ben Sammons.

## Installation

```bash
/plugin marketplace add git@github.com:Sammons/sammons-claude-marketplace.git
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `fancy-gdocs` | Google Docs DSL compiler — write docs in JSON, get formatted documents |

## Installing a Plugin

```bash
/plugin install fancy-gdocs
```

## Adding Your Own Marketplace

Fork this repo and edit `.claude-plugin/marketplace.json` to point to your own plugin repos.
